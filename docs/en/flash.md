# Flash Firmware

> Requires Chrome or Edge (Web Serial API).  
> Before flashing, make sure the board is connected via USB and V_IN is powered with 9–26 V.

> [!TIP] Enter Download Mode
> If the flash tool cannot connect, manually put the chip into download mode:  
> **Hold BOOT → press RESET → release RESET → release BOOT**  
> Then click **Connect & Flash** again.
