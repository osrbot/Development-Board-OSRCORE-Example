# 在线烧录

> 需要 Chrome 或 Edge 浏览器（支持 Web Serial API）。  
> 烧录前请确保开发板已通过 USB 接入电脑，且 V_IN 已接 9–26 V 电源。

> [!TIP] 进入下载模式
> 如果烧录工具提示无法连接，请手动让芯片进入下载模式：  
> **按住 BOOT 键 → 按一下 RESET 键 → 松开 RESET → 松开 BOOT**  
> 芯片进入下载模式后，再点击「连接并烧录」。
